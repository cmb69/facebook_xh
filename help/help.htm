Facebook_XH facilitates easy embedding of several Facebook services (such as like and share buttons) on your website.

Note that by nature of the Facebook services any visitor of your site who is logged in to Facebook can be tracked. If you want to better respect the privacy of your visitors, you may consider using Socialshareprivacy_XH instead of Facebook_XH.

Requirements

Facebook_XH is a plugin for CMSimple_XH. It requires CMSimple_XH ≥ 1.6 and PHP ≥ 5.

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory facebook/ to your server into CMSimple_XH's plugins/ directory.
  4. Set write permissions for the subdirectories css/, config/ and languages/.
back to top

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select "Facebook" from "Plugins".

You can change the default settings of Facebook_XH under "Config". Hints for the options will be displayed when hovering over the help icon with your mouse.

Localization is done under "Language". You can translate the character strings to your own language, or customize them according to your needs.

The look of Facebook_XH can be customized under "Stylesheet".

Usage

Like Buttons

To embed a like button on a page, use:

{{{facebook_like('http://example.com/');}}}

To embed a like button in the template, use:

<?php echo facebook_like('http://example.com/');?>

In either case you have to replace http://example.com/ with the URL you like to be liked. Simply omit the argument (i.e. use empty parentheses: ()) to use the current URL (i.e. the URL displayed in the browser's address bar).

You can embed multiple like buttons on a single page, independently of the URL they're referring to.

Share Buttons

To embed a share button on a page, use:

{{{facebook_share('http://example.com/');}}}

To embed a share button in the template, use:

<?php echo facebook_share('http://example.com/');?>

In either case you have to replace http://example.com/ with the URL you like to be shared. Simply omit the argument (i.e. use empty parentheses: ()) to use the current URL (i.e. the URL displayed in the browser's address bar).

You can embed multiple share buttons on a single page, independently of the URL they're referring to.

Limitations

Currently only embedding of like and share buttons is available.

The plugin is not (yet) registered on Facebook, so a warning will be shown in the browser's error console regarding the missing App Id. That shouldn't affect the proper functionality, though.
