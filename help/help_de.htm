Facebook_XH - ein CMSimple_XH Plugin

Facebook_XH ermöglicht das einfache Einbinden von verschiedenen Facebook-Diensten (wie Like- und Share-Buttons) auf Ihrer Website.

Beachten Sie, dass die Facebook-Dienste das Tracken von Besuchern Ihrer Website, die bei Facebook angemeldet sind, von Haus aus ermöglichen. Wenn Sie die Privatsphäre Ihrer Besucher besser schützen möchten, sollten Sie in Erwägung ziehen Socialshareprivacy_XH anstelle von Facebook_XH zu verwenden.

Voraussetzungen

Facebook_XH ist ein Plugin für CMSimple_XH. Es benötigt CMSimple_XH ≥ 1.6 und PHP ≥ 5.

Installation

Die Installation erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch. Im CMSimple_XH Wiki finden sie ausführliche Hinweise.

  1. Sichern Sie die Daten auf Ihrem Server.
  2. Entpacken Sie die ZIP-Datei auf Ihrem Computer.
  3. Laden Sie das gesamte Verzeichnis facebook/ auf Ihren Server in das plugins/ Verzeichnis von CMSimple_XH hoch.
  4. Vergeben Sie Schreibrechte für die Unterverzeichnisse css/, config/, und languages/.
zum Seitenanfang

Einstellungen

Die Konfiguration des Plugins erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch im Administrationsbereich der Homepage. Wählen Sie Plugins → Facebook.

Sie können die Original-Einstellungen von Facebook_XH unter "Konfiguration" ändern. Beim Überfahren der Hilfe-Icons mit der Maus werden Hinweise zu den Einstellungen angezeigt.

Die Lokalisierung wird unter "Sprache" vorgenommen. Sie können die Zeichenketten in Ihre eigene Sprache übersetzen, oder sie entsprechend Ihren Anforderungen anpassen.

Das Aussehen von Facebook_XH kann unter "Stylesheet" angepasst werden.

Verwendung

Facebook-Links

Um einen Facebook-Link auf einer Seite einzubinden, verwenden Sie:

{{{facebook_link('http://www.facebook.com/…');}}}

Um einen Facebook-Link im Template einzubinden, verwenden Sie:

<?php echo facebook_link('http://www.facebook.com/…');?>

In beiden Fällen müssen Sie das Auslassungszeichen (…) durch Ihren internen Facebook Benutzernamen ersetzen; anders ausgedrückt, müssen Sie Ihre Facebook-URL als Argument an die Funktion übergeben.

Sie können mehrere Facebook-Links auf einer einzelnen Seite einbinden.

zum Seitenanfang

Like-Buttons

Um einen Like-Button auf einer Seite einzubinden, verwenden Sie:

{{{facebook_like('http://example.com/');}}}

Um einen Like-Button im Template einzubinden, verwenden Sie:

<?php echo facebook_like('http://example.com/');?>

In beiden Fällen müssen Sie http://example.com/ durch die URL, die Sie "liken" lassen möchten, ersetzen. Lassen Sie einfach das Argument weg (d.h. verwenden Sie leere Klammern: ()), um die aktuelle URL zu verwenden (d.h. die URL, die in der Adressleiste des Browsers angezeigt wird).

Sie können mehrere Like-Buttons auf einer einzelnen Seite einbinden, unabhängig von der URL auf die sie sich beziehen.

Share-Buttons

Um einen Share-Button auf einer Seite einzubinden, verwenden Sie:

{{{facebook_share('http://example.com/');}}}

Um einen Share-Button im Template einzubinden, verwenden Sie:

<?php echo facebook_share('http://example.com/');?>

In beiden Fällen müssen Sie http://example.com/ durch die URL, die Sie teilen lassen möchten, ersetzen. Lassen Sie einfach das Argument weg (d.h. verwenden Sie leere Klammern: ()), um die aktuelle URL zu verwenden (d.h. die URL, die in der Adressleiste des Browsers angezeigt wird).

Sie können mehrere Share-Buttons auf einer einzelnen Seite einbinden, unabhängig von der URL auf die sie sich beziehen.

Beschränkungen

Derzeit ist nur das Einbinden von Like- und Share-Buttons möglich.

Das Plugin ist (noch) nicht bei Facebook registriert, so dass in der Fehlerkonsole des Browsers eine Warnung bezüglich der fehlenden App Id angezeigt wird. Dies sollte allerdings die ordnungsgemäße Funktion nicht gefährden.
